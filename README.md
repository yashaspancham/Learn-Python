# Learn Python: The Journey to Mastery

A structured, high-intensity learning path designed to achieve **Python mastery** and surpass LLM-level code generation. This repository tracks progress through a series of granular tasks, domain-specific projects, and a tiered roadmap from Beginner to God.

## 🎯 The Goal
To master Python's syntax, internals, and ecosystem to a degree where human-written code consistently outperforms LLM-generated skeletons in efficiency, readability, and structural integrity.

## 🛠 Project Structure
- **`Roadmap/`**: Tiered milestones (Beginner, Intermediate, Advanced, Ultra, God).
- **`Beginner/`**: 40 granular tasks focusing on fundamental logic and syntax.
- **`Projects/`**: 100 domain-specific projects to apply knowledge.
- **`style-guide.md`**: Mandatory standards for type hinting, documentation, and logic.

## 🧪 The "Zero Code Stub" Workflow
This project employs a unique interaction model with AI:
1. **LLM Role**: Provides documentation, syntax examples, and `# TODO` requirements.
2. **Strict Constraint**: No skeleton functions, boilerplate, or code stubs are provided.
3. **User Role**: Writes every line of code from scratch, adhering to the project's strict style guide.
4. **Validation**: Code is reviewed for PEP 8 compliance, Google-style docstrings, and mandatory type hinting.

## 📜 Coding Standards
All code in this repository follows the `style-guide.md`:
- **Mandatory Type Hinting**: Using the `typing` module for all interfaces.
- **Google Style Docstrings**: Detailed documentation for every module and function.
- **Clean Architecture**: Functions under 20 lines, guard clauses, and pure logic.
- **PEP 8**: Strict adherence to formatting and naming conventions.
